The Falcon
==========

A child theme for Twenty Fourteen with all content centered, user-configurable gradient background, fancier topbar and a slideout sidebar on mobile.

More Information
================
* Theme: http://jpwp.me/thefalcon
* Author: http://JoshPress.net
